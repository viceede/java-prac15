# HTTP-сервер с калькулятором

## Описание проекта
Простой HTTP-сервер на Java, реализующий функционал калькулятора для выполнения арифметических операций.

## Функционал
1. Выполнение арифметических операций:
    - Сложение (+)
    - Вычитание (-)
    - Умножение (*)
    - Деление (/)
2. Информационная страница о сервере
3. Главная страница с примерами использования
